# Cenexis Bio Stealth Website

This is the static landing page for Cenexis Bio.

## How to Host on GitHub Pages

1. **Create a Repository:**
   - Go to GitHub and create a new public repository (e.g., `cenexis-site`).

2. **Upload Files:**
   - Upload `index.html`, `styles.css`, and `script.js` to the root of the repository.

3. **Activate GitHub Pages:**
   - Go to the repository **Settings**.
   - Click on **Pages** (in the left sidebar).
   - Under **Build and deployment** > **Branch**, select `main` (or `master`) and ensure the folder is `/(root)`.
   - Click **Save**.

4. **View Site:**
   - After a minute, your site will be live at `https://[your-username].github.io/cenexis-site/`.
